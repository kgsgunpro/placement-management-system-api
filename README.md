# Placement Management System API

A production-style backend application built using FastAPI to manage the complete campus placement workflow.

## Vision

This project is being developed as if it were a real startup product rather than a college project.

The objective is to learn backend engineering by building production-quality software.

## Planned Features

- Student Management
- Company Management
- Job Posting Management
- Application Tracking
- Interview Scheduling
- Offer Management
- Authentication & Authorization
- Dashboard & Analytics

## Tech Stack

- Python
- FastAPI
- PostgreSQL (Planned)
- SQLAlchemy (Planned)
- JWT Authentication (Planned)
- Docker (Planned)
- Pytest (Planned)
